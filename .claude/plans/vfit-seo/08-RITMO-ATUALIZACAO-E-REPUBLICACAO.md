# 08 — Pauta de Atualização e Republicação

## Objetivo

Manter artigos vivos, com sinais de atualização real para motores de busca e IA.

## Regras de atualização

### A cada 30 dias

- revisar títulos e `description` dos artigos com CTR abaixo da média
- revisar primeiro bloco (resposta direta) para intenção de busca atual
- incluir ao menos 1 novo link interno estratégico

### A cada 60 dias

- atualizar comparativos e tabelas
- revisar FAQ para linguagem natural mais recente
- revisar CTA principal por ICP

### A cada 90 dias

- republicar os artigos pilar com melhoria substancial
- atualizar `modifiedISO` no índice
- validar snippet de rich results

## Critérios para republicar

Republicar quando houver ao menos 2 sinais:

- queda >20% de cliques
- perda de posição para keyword principal
- conteúdo desatualizado por produto/mercado
- oportunidade de cluster com nova página money

## Checklist de republicação

- [ ] atualizar dados/tabelas/exemplos
- [ ] adicionar links internos novos
- [ ] revisar FAQ
- [ ] validar schema
- [ ] atualizar `modifiedISO`
- [ ] registrar no tracking
