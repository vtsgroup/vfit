## Objetivo
Transformar o VFIT em produto **ultra premium**, seguro e escalável, com operação em produção de nível global.

## Regras fixas do plano
1. Não alterar navbar mobile atual.
2. Não alterar botão do robô (copy link).
3. Toda entrega em lotes pequenos e auditáveis.
4. Cada lote precisa: escopo, risco, rollout, rollback, métricas.
