## 📊 Project Overview

**VFIT** is a full-stack SaaS platform designed for personal trainers (B2B) and fitness students (B2C) to manage workout plans, track progress, handle payments, and leverage AI for content generation.

### Key Features
- **Workout Management:** Create, assign, track workouts with exercise libraries
- **AI-Powered:** Generate workout plans and content using Replicate API & Cloudflare Workers AI
- **Gamification:** XP system, streaks, daily goals, achievements
- **Assessment Tools:** Physical assessments with photo analysis and composition tracking
- **Payment Processing:** Full Asaas integration (PIX, boleto, credit card)
- **Affiliate Program:** Referral system with commission tracking
- **Chat System:** Direct messaging between trainers and students
- **Notifications:** Multi-channel (push, email, in-app, WhatsApp)
- **Analytics:** Progress tracking, performance reports, financial dashboards
- **PWA & Mobile:** Progressive web app + iOS/Android via TWA/APK

```
   ├─ Input: email, password, turnstile_token
   ├─ Validate: Turnstile → bcryptjs compare
   ├─ Generate: access_token (1h TTL) + refresh_token (30d)
   ├─ Store in KV: { refresh_token, user_id, expires_at }
   └─ Response: { access_token, refresh_token, expires_at, user }

2. REQUEST WITH TOKEN
   GET /api/v1/auth/me
   ├─ Header: Authorization: Bearer <access_token>
   ├─ Middleware: authMiddleware() decodes token
   │  └─ Uses: Web Crypto HMAC-SHA256 (JWT_SECRET)
   ├─ Extract: user_id, user_type from token payload
   ├─ Response: Full user + profile data

3. TOKEN REFRESH
   POST /api/v1/auth/refresh
   ├─ Input: refresh_token
   ├─ Validate: Check KV for token + expiry
   ├─ If valid: Generate new access_token
   ├─ Auto-deduped: Multiple concurrent refresh → single query
   └─ Response: { access_token, expires_at }

4. LOGOUT
   POST /api/v1/auth/logout
   ├─ Header: Authorization: Bearer <token>
   ├─ Action: Delete refresh_token from KV (blacklist)
   └─ Response: 204 No Content

5. TWO-FACTOR AUTH (Optional)
   POST /api/v1/auth/2fa/setup
   ├─ Generate: TOTP secret (qrcode for user)
   ├─ Store: Encrypted in user.metadata
   
   POST /api/v1/auth/2fa/verify
   ├─ Input: 6-digit TOTP code
   ├─ Validate: speakeasy library
   └─ Enable: Flag in users table
```

### Token Signing & Verification

```typescript
// Sign (backend)
const token = crypto.jwt.sign({ user_id, user_type, role }, JWT_SECRET, 'HS256')
// TTL embedded: `exp: Math.floor(Date.now() / 1000) + 3600`

// Verify (middleware)
const payload = crypto.jwt.verify(token, JWT_SECRET, 'HS256')
// Checks: signature + expiry

// Frontend (Zustand)

// Persists to localStorage, auto-hydrated on page load
```

**Example (R$ 100 charge):**
- Gross: R$ 100.00
- Asaas fee (3.49%): -R$ 3.49
- Platform commission (5%): -R$ 5.00

## 🛠️ Key Technologies & Design Decisions

### Why This Stack?

#### Frontend
- **Next.js 15:** Static export for CF Pages (near-zero latency), partial ISR via edge middleware
- **Tailwind v4:** Reduced CSS bundle (1.2MB → 300KB), native CSS variables, @layer benefits
- **Zustand:** Lightweight state (no boilerplate), localStorage persistence built-in, tiny bundle
- **TanStack Query:** Server state sync without Redux, auto caching, background refresh
- **Framer Motion:** Smooth animations without layout thrashing, GPU-accelerated transforms

#### Backend
- **Cloudflare Workers:** Sub-50ms TTFB globally, no cold starts, built-in DDoS protection
- **Hono.js:** Lightweight (3KB core), Cloudflare native, faster than Express for edge
- **Zod:** Runtime validation (catches DB errors), TypeScript inference, tiny bundle
- **Raw SQL:** No ORM lock-in, full control over queries, better perf for analytics

#### Database
- **Neon PostgreSQL:** Serverless, HTTP API (compatible with edge), auto-scaling, backup included
- **D1 (SQLite):** Cold data cache, no connection pooling overhead, native CF binding
- **KV for Cache:** Near-instant access, geo-distributed, perfect for session storage

#### Integrations
- **Asaas:** Brazil-only payment processor, PIX native (instant settlement), no chargebacks
- **Replicate API:** Cheap AI inference ($0.001/image), supports fine-tuned models
- **Cloudflare Workers AI:** Free tier (10k neurons/day), native integration, no egress fees
- **OneSignal:** Multi-channel notifications (push, email, SMS), targeting by segment

### Architectural Patterns

#### Error Handling
