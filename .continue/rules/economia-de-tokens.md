---
name: economia-de-tokens
description: Regras para minimizar consumo de tokens
alwaysApply: true
---

- Seja direto e conciso — sem introduções, sem resumos no final, sem frases óbvias
- Nunca repita código que já existe no contexto — mostre apenas o bloco alterado
- Não gere documentação, JSDoc ou comentários a menos que explicitamente solicitado
- Prefira comentários inline a blocos de comentário
- Só use tool calls quando forem estritamente necessários para completar a tarefa
- Não liste o que foi feito após executar uma ação — apenas execute
- Evite exemplos hipotéticos quando o código real já está disponível no contexto
- Nunca reescreva arquivos inteiros; edite apenas o trecho relevante
