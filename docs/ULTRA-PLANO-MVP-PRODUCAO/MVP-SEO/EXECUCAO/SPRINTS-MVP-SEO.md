## Sprint 00 — Fundação técnica (Semana 0)

### Objetivo
Deixar toda base técnica pronta para escalar conteúdo e páginas sem retrabalho.

### Escopo
- setup SEO técnico (metadata factory, schemas, sitemap)
- setup analytics (GA4/GTM/eventos base)
- setup A/B test base
- setup PWA mínimo
- setup docs operacionais de execução

### Entregas obrigatórias
- homepage com schemas globais
- /planos e /download publicadas
- robots.txt + llms.txt revisados
- Cloudflare liberando crawlers de IA
- baseline de Core Web Vitals

### Critério de saída
- 100% checklist Sprint 00 marcado
- sem blocker crítico de infra
